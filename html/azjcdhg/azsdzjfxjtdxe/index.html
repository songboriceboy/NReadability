安卓四大组件复习监听短信（二）

 这一节 用一个demo 复习广播，

思路：需要一个APP，监听系统的短信，不需要界面。只需要一个广播

好啦 首先是新建一个广播了SmsReciver.java
 根据谷歌原则，四大组件都要注册

        <receiver android:name="com.example.SmsReciver.SmsReciver">
            <intent-filter android:priority="1000">
                <action android:name="android.provider.Telephony.SMS_RECEIVED"/>
            </intent-filter>
        </receiver>

由于上一节 我已经仔细讲过配置文件的作用了  这一节只加入了  优先级 priority   1000是最高的

后面加入要监听的广播 短信 接收  android:name="android.provider.Telephony.SMS_RECEIVED"

 记得加上权限

其实你可以发现 四大组件的通讯人 是intent的  他四处出现，所以我们可以从intent里面获取一个map集合的短信内容

    Object[] pdus = (Object[]) intent.getExtras().get("pdus");

然后就是获取 短信内容和联系人 已经短信拦截 发送短信  记得加权像



package com.example.SmsReciver;

import android.content.BroadcastReceiver;
import android.content.Context;
import android.content.Intent;
import android.telephony.SmsManager;
import android.telephony.SmsMessage;

public class SmsReciver extends BroadcastReceiver {

	@Override
	public void onReceive(Context context, Intent intent) {
		Object[] pdus = (Object[]) intent.getExtras().get("pdus");
		for (Object pdu : pdus) {
			SmsMessage smsMessage = SmsMessage.createFromPdu((byte[]) pdu);
			String body = smsMessage.getMessageBody();
			String sender = smsMessage.getOriginatingAddress();
			//可以通过邮箱发送
			System.out.println("body:" + body);
			System.out.println("sender:" + sender);
			if ("5556".equals(sender)) {
				// 短信的拦截
				abortBroadcast();
				SmsManager smsManager = SmsManager.getDefault();
				smsManager.sendTextMessage(sender, null, "I already to love jim,your must leave me;", null, null);
			}
		}

	}

}
